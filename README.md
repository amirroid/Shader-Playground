# Shading

A real-time GLSL shader editor for Android.

## Features

- Write and edit fragment shaders directly on your device
- Live preview with OpenGL ES 2.0 rendering
- Uniform parameter sliders (`uParam0`–`uParam3`) for interactive shader tuning
- Shader compilation error display
- Dark IDE-style interface

## Built With

- Kotlin
- Jetpack Compose + Material 3
- OpenGL ES 2.0

## Requirements

- Android 7.0 (API 24) or higher

## Building

Open the project in Android Studio and run on a device or emulator.

```bash
./gradlew assembleDebug
```
